# AlphaForge 项目计划

## 一、项目概述

### 1.1 项目背景

AlphaForge 是一个基于真实股市数据的虚拟交易平台，旨在为投资者提供一个零风险的学习和练习环境。用户可以模拟真实交易体验，学习投资策略，同时后期将集成AI分析功能辅助决策。

### 1.2 项目目标

- 提供接近真实的股票交易体验
- 支持多档初始本金选择，适应不同用户需求
- 实时计算持仓盈亏，培养投资意识
- 通过LLM提供智能投资分析和建议

### 1.3 核心价值

| 价值点 | 说明 |
|--------|------|
| 零风险练习 | 真实市场数据，虚拟资金，无实际损失 |
| 多场景适配 | 支持不同资金量级（10万-1000万美元） |
| 实时反馈 | 即时计算盈亏，直观展示投资效果 |
| AI辅助决策 | 后期接入LLM，提供专业分析 |

### 3.3 股票数据源：Alpha Vantage

本项目采用 **Alpha Vantage** 作为官方数据源，提供全球股票、外汇、加密货币等金融数据。

#### API Key
```
6NM9RLOVI80BJ8WY
```

#### 免费额度限制
| 限制项 | 额度 |
|--------|------|
| 每日请求 | 25次 |
| 每分钟请求 | 5次 |
| 数据延迟 | 约5分钟 |

#### 核心API端点

| 端点 | 功能 | 示例 |
|------|------|------|
| `TIME_SERIES_INTRADAY` | 日内K线数据 | 1min/5min/15min/30min/60min |
| `TIME_SERIES_DAILY` | 日K线数据 | 20+年历史数据 |
| `GLOBAL_QUOTE` | 实时报价 | 当前价格、涨跌幅 |
| `SYMBOL_SEARCH` | 股票搜索 | 根据关键词查找股票代码 |

#### API调用示例

```http
# 获取实时报价
GET https://www.alphavantage.co/query?function=GLOBAL_QUOTE&symbol=AAPL&apikey=YOUR_KEY

# 获取日线数据
GET https://www.alphavantage.co/query?function=TIME_SERIES_DAILY&symbol=AAPL&apikey=YOUR_KEY

# 获取5分钟K线
GET https://www.alphavantage.co/query?function=TIME_SERIES_INTRADAY&symbol=AAPL&interval=5min&apikey=YOUR_KEY

# 搜索股票
GET https://www.alphavantage.co/query?function=SYMBOL_SEARCH&keywords=Tesla&apikey=YOUR_KEY
```

#### 额外功能支持

Alpha Vantage 还提供以下高级功能（可在后期扩展）：

- **基本面数据**: 财报、资产负债表、现金流
- **技术指标**: SMA、EMA、MACD、RSI、布林带等 50+ 指标
- **新闻情绪**: 股票相关新闻与情感分析
- **经济指标**: GDP、CPI、利率、失业率等

详细文档请参考 [APIdoc.md](./APIdoc.md)

### 3.4 数据缓存策略

由于 Alpha Vantage 有请求限制，必须采用积极的缓存策略：

| 数据类型 | 缓存时长 | 理由 |
|----------|----------|------|
| 实时报价 | 1-5分钟 | API数据本身有延迟 |
| 日线数据 | 1天 | 每日更新一次即可 |
| 股票搜索 | 7天 | 股票代码很少变更 |
| 历史K线 | 永久 | 历史数据不变 |

## 六、数据模型设计

### 6.1 核心实体

```
用户 (Users)
  ├─ 账户 (Accounts)
  │   ├─ 持仓 (Positions)
  │   └─ 订单 (Orders)
  └─ 操作日志 (ActivityLogs)

  └─ 行情缓存 (Quotes)
```

### 6.2 数据表概要

| 表名 | 主要字段 | 说明 |
|------|----------|------|
| users | id, email, password | 用户信息 |

| positions | id, account_id, symbol, shares, avg_cost | 持仓信息 |
| orders | id, account_id, symbol, type, shares, price, status | 交易记录 |
| stock_quotes | symbol, price, change, volume, updated_at | 行情缓存 |
| activities | id, user_id, action, details, timestamp | 操作日志 |
